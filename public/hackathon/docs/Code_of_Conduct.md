# Código de Conducta — Hackathon Bitcoin México

## Principios Fundamentales

El Hackathon Bitcoin México se rige por los principios de Bitcoin: descentralización, transparencia, respeto y soberanía individual.

## Comportamiento Esperado

- **Respeto mutuo**: Todos los participantes, mentores y jueces merecen respeto sin importar experiencia, género, orientación, etnia o nacionalidad.
- **Colaboración sobre competencia**: Bitcoin es dinero descentralizado. Ayuda a otros equipos cuando puedas.
- **Código abierto**: Todos los proyectos deben ser open source.
- **Cero toxicidad**: No se toleran insultos, acoso ni discriminación de ningún tipo.
- **Privacidad**: Respetamos la privacidad de cada participante. No compartas información personal sin consentimiento.

## Prohibiciones

- Discriminación, acoso o amenazas
- Código propietario o con licencias restrictivas
- Bots o herramientas automatizadas que den ventaja injusta
- Uso de código generado por IA sin atribución y divulgación

## Reportes

Si experimentas una violación del código de conducta:
- Contacta a los organizadores: hackathon@aceptabitcoin.org
- Habilita el canal de reportes anónimos en Discord

**Todos los participantes aceptan este código al registrarse.**
