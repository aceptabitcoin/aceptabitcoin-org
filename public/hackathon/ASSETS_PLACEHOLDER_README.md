# Hackathon Assets

This directory contains logos, images, and assets for the Hackathon Bitcoin México module.

## Structure

- `logos/` — Sponsor logos, project branding, event identity
- `assets/` — Illustrations, backgrounds, decorative assets
- `docs/` — PDF documentation, rulebooks, guides

## Notes

- All SVG logos should use `currentColor` or transparent backgrounds
- Images should be optimized (WebP preferred for photos, SVG for graphics)
- Maximum file size: 500KB per asset
